# Implementación de Filtros Digitales

Este proyecto consiste en el diseño e implementación de filtros digitales utilizando Python.

## Descripción

Se generó una señal con dos frecuencias (50 Hz y 200 Hz) y ruido agregado. Posteriormente, se aplicaron tres filtros:

- Filtro pasa bajos  
- Filtro pasa altos  
- Filtro pasa bandas  

## Resultados

Los filtros permitieron eliminar o conservar ciertas frecuencias de la señal original.

## Tecnologías utilizadas

- Python  
- NumPy  
- Matplotlib
